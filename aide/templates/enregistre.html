Enregistrer

Comment enregistrer les données du bus CAN

  1. Cliquez sur "Ouvrir un fichier texte" dans la barre d'outils.
    • Entrez le nom du fichier .txt.
    • Cliquer sur 'Enregistrer'.
  2. Cochez la case à cocher "Enregister le bus CAN sur le fichier".
  3. Cliquez sur le bouton marqué "Open" dans la barre d'outils.
  4. Cliquez sur le bouton de lecture dans la barre d'outils.
    • Vous voyez le nombre de trames enregistrées sur l'écran
  5. Vous arrêtez quand vous voulez.
  6. Cliquez sur bouton en forme de livre sur la barre d'outils pour visualiser le fichier

Erreur : Si le bouton de lecture réapparait, c'est que vous n'avez pas de trame, vérifiez que vous êtes bien raccordé au bus NMEA 2000 et que celui-ci est bien alimenté et qu'il y a au moins un instrument de raccordé.
  • Voyez un exemple du fichier rempli.
  • Vous voyez le TimeStamp en ms, l'ID, la longueur et les datas.

Formats supportés

  • Fichiers .txt
